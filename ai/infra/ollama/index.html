Ollama

安装部署

Docker
# 1.拉取容器
docker run -d \
  --publish 11434:11434 \
  --volume ollama:/root/.ollama \
  --name=ollama \
  ollama/ollama

# 2.执行命令
docker exec -it ollama ollama run deepseek-r1

The official Ollama Docker image ollama/ollama is available on Docker Hub.

Linux
curl -fsSL https://ollama.com/install.sh | sh
