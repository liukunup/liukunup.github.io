HF-Mirror

约 120 字小于 1 分钟

2025-10-13

Hugging Face 无法访问？

HF-Mirror作为一个公益项目，致力于帮助国内AI开发者快速、稳定的下载模型、数据集。

使用说明

  1. 安装工具
sudo apt install -y aria2 jq
  1. 下载hfd
wget https://hf-mirror.com/hfd/hfd.sh
chmod a+x hfd.sh
  1. 设置环境变量
  • Linux
export HF_ENDPOINT=https://hf-mirror.com
  • Windows
$env:HF_ENDPOINT = "https://hf-mirror.com"

4.1 下载模型

./hfd.sh gpt2

4.2 下载数据集

./hfd.sh wikitext --dataset
