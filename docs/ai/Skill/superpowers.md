## Superpowers

[GitHub](https://github.com/obra/superpowers) — ⭐ 229k

> An agentic skills framework & software development methodology that works.

Superpowers 是一套面向 AI 编码助手的技能框架和软件开发方法论，提供可组合的 prompt 技能，覆盖从需求分析、设计、实现、测试到代码审查的完整开发流程。

### 核心工作流

1. **brainstorming** — 需求分析与设计探索，生成设计文档
2. **using-git-worktrees** — 创建隔离开发环境
3. **writing-plans** — 分解任务，每项 2-5 分钟
4. **subagent-driven-development / executing-plans** — 子代理驱动的迭代开发
5. **test-driven-development** — 红-绿-重构循环
6. **requesting-code-review** — 逐项审查
7. **finishing-a-development-branch** — 分支收尾

### 支持的编码代理

Claude Code、Codex CLI、Codex App、Factory Droid、Gemini CLI、**OpenCode**、Cursor、GitHub Copilot CLI
