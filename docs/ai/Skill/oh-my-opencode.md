## Oh My OpenCode / Oh My OpenAgent

[GitHub](https://github.com/code-yeongyu/oh-my-openagent) — ⭐ 93k

> omo; the best agent harness — previously oh-my-opencode

Oh My OpenCode 是 OpenCode 生态中最流行的插件，提供多模型编排、异步并行代理执行、LSP/AST 确定性重构、内置 MCP 等能力。

### 核心特性

| 特性 | 说明 |
|------|------|
| **Ultrawork 模式** | 关键词触发并行执行、LSP 集成、上下文管理、自动任务分解 |
| **Specialist Agents** | 命名角色如 @oracle（架构）、@librarian（研究/文档） |
| **多模型编排** | 每类任务自动分配最优模型 |
| **内置 MCP** | Web 搜索、官方文档查询、GitHub 代码搜索 |
| **Hook 系统** | 兼容 Claude Code hooks |

### 安装

```bash
bunx oh-my-opencode install
# 或
npx oh-my-opencode install
```

或在 OpenCode 中执行：`Install and configure oh-my-opencode by following the instructions here: https://raw.githubusercontent.com/code-yeongyu/oh-my-openagent/refs/heads/dev/docs/guide/installation.md`
