## 快速部署

::tdesign:logo-github-filled:: [Dify](https://github.com/langgenius/dify.git)

::: info
**系统要求**
- CPU >= 2 Core
- RAM >= 4 GiB
:::

```shell
# 1.克隆仓库
git clone https://github.com/langgenius/dify.git

# 2.配置参数
cd dify/docker
cp .env.example .env

# 3.启动服务
docker compose up -d
```
