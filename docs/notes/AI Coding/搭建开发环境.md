## Node.js®

https://nodejs.org/zh-cn/download

```bash
# 下载并安装 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash

# 代替重启 shell
\. "$HOME/.nvm/nvm.sh"

# 下载并安装 Node.js
nvm install 24

# 验证 Node.js 版本
node -v # Should print "v24.16.0".

# 验证 npm 版本
npm -v # Should print "11.13.0".

# 下载并安装 pnpm
corepack enable pnpm
# 验证 pnpm 版本
pnpm -v

# 下载并安装 Yarn
corepack enable yarn
# 验证 Yarn 版本
yarn -v
```
