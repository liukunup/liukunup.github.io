- **[MarkItDown](https://github.com/microsoft/markitdown)** ::codicon:mcp::

> MarkItDown 是一个轻量级 Python 工具，专注于将多种文件格式转换为 Markdown，供 LLM 和文本分析使用。
> 
> 其特点是保留文档结构（标题、列表、表格、链接等），输出主要面向文本分析工具而非人类阅读。
