## 前言

在信息爆炸的时代，社交媒体的时间线往往被算法控制，你看到的内容取决于平台想让你看到什么。**feeeed**（四连 e）是一款专注于让你掌控自己信息源的 iOS 应用——构建属于你自己的、无广告的、个性化的资讯聚合阅读器。

> "Stay informed without losing sanity. A feed curated by you, not the social-media mob."

## 应用简介

**feeeed** 由独立开发者 Nate Parrott 开发，是一款免费的 iOS/iPadOS 阅读器应用（macOS via compatibility mode）。它不仅仅是一个 RSS 阅读器，而是将多种内容源统一到一个流畅的时间线中。

### 核心特点

| 特性 | 说明 |
|------|------|
| **隐私优先** | 100% 本地运行，无服务器、无账户、不收集数据 |
| **多源订阅** | RSS、YouTube、Reddit、Hacker News、Mastodon、Threads、TikTok 等 |
| **AI 摘要** | 免费文章 AI 智能摘要 |
| **阅读模式** | 快速、专注、可自定义的阅读器模式 |
| **离线支持** | 下载文章离线阅读 |
| **小组件** | iOS 主屏幕小组件 |
| **自定义提醒** | 每日步数目标、健康数据、照片回忆等 |

### 支持的平台

- iPhone (iOS 16.0+)
- iPad (iPadOS 16.0+)
- Mac (macOS 13.0+, Apple M1+)
- Apple Vision (visionOS 1.0+)

## 功能亮点

### 1. 订阅任意网站

通过 RSS 订阅任何网站的内容，也可以直接从 Safari 分享菜单添加网站关注（v17+ 新功能）。

### 2. 多平台内容聚合

除了传统 RSS，feeeed 还支持：

- **YouTube 频道** - 订阅你喜欢的频道
- **Subreddits** - 关注 Reddit 社区
- **Hacker News** - 获取技术新闻
- **Mastodon/Threads** - 关注社交动态
- **Newsletter** - Gmail 邮件 newsletter（已暂停，Google 停用了相关 API）

### 3. 个性化卡片

feeeed 可以将任何网站的一部分转换为实时卡片，方便追踪特定内容区域。

### 4. 健康与生活集成

- **步数统计** - 追踪每日步数和 10,000 步目标进度
- **照片回忆** - 偶尔展示旧照片
- **生日提醒** - 联系人生日提醒
- **自定义提醒** - 创建个性化提醒卡片

### 5. AI 机器人

内置 AI 助手可以根据主题推荐订阅源，例如询问"有哪些关于足球的博客或 newsletter"，机器人会给出建议。

## 使用技巧

### 1. 从 Safari 添加订阅

1. 在 Safari 中打开目标网站
2. 点击分享按钮
3. 选择 **feeeed**
4. 点击 **Follow** 关注

### 2. 导入 OPML

如果你是其他 RSS 阅读器的用户，可以导出 OPML 文件并在 feeeed 中导入，快速迁移订阅源。

### 3. 调整信息流算法

- **Best Variety** (默认) - 算法推荐，平衡内容多样性
- **Chronological** - 完全按时间顺序
- 可以在底部标签栏固定 **Latest** 快速切换

### 4. 离线阅读

在有网络时下载数十篇文章，无网时随时阅读，适合通勤或旅行。
