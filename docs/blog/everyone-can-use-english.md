# 人人都能用英语：34.8k Stars 的 AI 英语学习开源项目

> "AI 是当今世界上最好的外语老师，Enjoy 做 AI 最好的助教。"

<!-- more -->

## 一、项目概览

[ZuodaoTech/everyone-can-use-english](https://github.com/ZuodaoTech/everyone-can-use-english) 是 GitHub 上最受欢迎的英语学习开源项目之一，目前拥有 **34.8k Stars** 和 **4.9k Forks**。

该项目由两部分组成：
- **Enjoy App** — AI 驱动的英语学习应用
- **1000 小时训练体系** — 系统的英语习得方法论

---

## 二、Enjoy App 核心功能

### 2.1 网页版

访问 [https://enjoy.bot](https://enjoy.bot) 直接使用。Enjoy 提供四大核心功能：

| 功能 | 说明 |
|------|------|
| **视频学习** | 沉浸式视频内容学习 |
| **电子书阅读** | 带 AI 辅助的阅读理解 |
| **闪卡记忆** | 间隔重复记忆训练 |
| **课程学习** | 结构化课程体系 |

### 2.2 浏览器插件

Enjoy 浏览器插件已上线，支持 **YouTube** 和 **Netflix** 平台。在观看视频时即可获得实时辅助：

- 实时字幕与翻译
- 词汇高亮与解释
- 跟读练习

可在 [Chrome Web Store](https://chromewebstore.google.com/detail/enjoy-echo/hiijpdndbjfnffibdhajdanjekbnalob) 安装。

### 2.3 桌面版

新版桌面版即将发布，将对网页版进行套壳和增强。

## 三、1000 小时训练体系

该项目配套的 [1000h.org](https://1000h.org) 提供了完整的英语习得训练路径：

### 核心模块

| 模块 | 内容 |
|------|------|
| [简要说明](https://1000h.org/intro.html) | 训练体系总览 |
| [训练任务](https://1000h.org/training-tasks/kick-off.html) | 每日训练任务 |
| [语音塑造](https://1000h.org/sounds-of-american-english/0-intro.html) | 美音发音系统训练 |
| [大脑内部](https://1000h.org/in-the-brain/01-inifinite.html) | 语言习得原理 |
| [自我训练](https://1000h.org/self-training/00-intro.html) | 自学方法指导 |

### 核心理念

1000 小时训练法强调：
1. **大量输入** — 通过听力和阅读建立语感
2. **语音塑造** — 从一开始就练习正确发音
3. **持续输出** — 鼓励说和写的实践

## 六、项目特色

### 6.1 AI 原生设计

Enjoy 不是将 AI 强塞进传统学习流程，而是**从一开始就围绕 AI 能力设计**：

- AI 作为**真正的老师**而非辅助工具
- 个性化学习路径推荐
- 实时语音纠正与反馈

### 6.2 开放透明

作为开源项目：
- 训练方法论完全公开
- 鼓励社区贡献
- 持续迭代更新

### 6.3 理论与实践结合

既有扎实的语言习得理论支撑（1000 小时体系），又有现代化的产品体验（Enjoy App）。
