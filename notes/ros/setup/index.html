搭建开发环境

约 895 字大约 3 分钟

2026-05-17

ROS: Home

Ubuntu 24.04

Ubuntu (deb packages)

设置语言环境

确保 locale 支持 UTF-8

locale  # check for UTF-8

sudo apt update && sudo apt install locales
sudo locale-gen en_US en_US.UTF-8
sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8
export LANG=en_US.UTF-8

locale  # verify settings

启用软件源

  1. 确保 Universe 仓库已启用
sudo apt install software-properties-common
sudo add-apt-repository universe
  1. 安装 ros-apt-source 包
sudo apt update && sudo apt install curl -y
export ROS_APT_SOURCE_VERSION=$(curl -s https://api.github.com/repos/ros-infrastructure/ros-apt-source/releases/latest | grep -F "tag_name" | awk -F'"' '{print $4}')
curl -L -o /tmp/ros2-apt-source.deb "https://github.com/ros-infrastructure/ros-apt-source/releases/download/${ROS_APT_SOURCE_VERSION}/ros2-apt-source_${ROS_APT_SOURCE_VERSION}.$(. /etc/os-release && echo ${UBUNTU_CODENAME:-${VERSION_CODENAME}})_all.deb"
sudo dpkg -i /tmp/ros2-apt-source.deb
  1. 安装 ros-dev-tools 包
sudo apt update && sudo apt install ros-dev-tools

安装 ROS 2

sudo apt update
sudo apt upgrade
  • desktop（推荐） 完整安装，包含 ROS、RViz、演示和教程
sudo apt install ros-jazzy-desktop
  • ros-base 仅通信库、消息包和命令行工具
sudo apt install ros-jazzy-ros-base

设置环境变量

source /opt/ros/jazzy/setup.bash

验证安装

# 终端 1
source /opt/ros/jazzy/setup.bash
ros2 run demo_nodes_cpp talker

# 终端 2
source /opt/ros/jazzy/setup.bash
ros2 run demo_nodes_py listener

卸载

sudo apt remove '~nros-jazzy-*' && sudo apt autoremove

sudo apt remove ros2-apt-source
sudo apt update
sudo apt autoremove
sudo apt upgrade # Consider upgrading for packages previously shadowed.

Windows 10

安装 pixi

ROS 2 使用 pixi 管理依赖。按照 pixi.sh 指引安装。

安装完成后重新打开命令提示符，确保 pixi 在 PATH 中。

下载 ROS 2

Rolling Ridley 不提供正式版二进制包，需下载夜间预发布版：

  1. 下载 ros2-package-windows-AMD64.zip
  2. 解压到 C:\dev
  3. 将文件夹重命名为 C:\dev\rolling

安装依赖

cd C:\dev\rolling
pixi install
pixi run python preinstall_setup_windows.py

设置环境变量

cd C:\dev\rolling
call C:\dev\rolling\local_setup.bat

相关信息

每次新打开命令提示符都需要执行此命令。

验证安装

# 终端 1
ros2 run demo_nodes_cpp talker

# 终端 2
ros2 run demo_nodes_py listener

卸载

rmdir /s /q C:\dev\rolling

macOS

macOS 不支持二进制包，需要源码编译安装。

系统要求

支持 macOS Mojave (10.14) 及以上。

安装 Xcode

从 App Store 安装 Xcode，然后安装 Command Line Tools：

xcode-select --install
sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer
sudo xcodebuild -license

安装 Homebrew

按照 brew.sh 指引安装，然后安装依赖：

brew install asio assimp bison bullet cmake console_bridge cppcheck \
  cunit eigen freetype graphviz opencv openssl orocos-kdl pcre poco \
  pyqt@5 python qt@5 sip spdlog tinyxml2

设置环境变量

echo "export OPENSSL_ROOT_DIR=$(brew --prefix openssl)" >> ~/.zshrc
export CMAKE_PREFIX_PATH=$CMAKE_PREFIX_PATH:$(brew --prefix qt@5)
export PATH=$PATH:$(brew --prefix qt@5)/bin

安装 Python 依赖

python3 -m pip install --upgrade pip

python3 -m pip install -U \
  --config-settings="--global-option=build_ext" \
  --config-settings="--global-option=-I$(brew --prefix graphviz)/include/" \
  --config-settings="--global-option=-L$(brew --prefix graphviz)/lib/" \
  argcomplete catkin_pkg colcon-common-extensions coverage \
  cryptography empy flake8 flake8-blind-except==0.1.1 flake8-builtins \
  flake8-class-newline flake8-comprehensions flake8-deprecated \
  flake8-import-order flake8-quotes \
  importlib-metadata jsonschema lark==1.1.1 lxml matplotlib mock mypy==0.931 netifaces \
  psutil pydot pygraphviz pyparsing==2.4.7 \
  pytest-mock rosdep rosdistro setuptools==59.6.0 vcstool

禁用 SIP

macOS 10.11+ 默认启用 SIP。为使进程能继承动态链接器环境变量，需禁用 SIP。

获取源码

mkdir -p ~/ros2_rolling/src
cd ~/ros2_rolling
vcs import --input https://raw.githubusercontent.com/ros2/ros2/rolling/ros2.repos src

编译

cd ~/ros2_rolling/
colcon build --symlink-install --packages-skip-by-dep python_qt_binding

设置环境变量

. ~/ros2_rolling/install/setup.zsh

验证安装

# 终端 1
ros2 run demo_nodes_cpp talker

# 终端 2
ros2 run demo_nodes_py listener

卸载

rm -rf ~/ros2_rolling

快速入门

配置环境

# Ubuntu
source /opt/ros/rolling/setup.bash

# macOS
. ~/ros2_rolling/install/setup.zsh

常用命令

# 列出所有包
ros2 pkg list

# 运行节点
ros2 run <package_name> <executable_name>

# 列出运行中的节点
ros2 node list

创建工作空间

mkdir -p ~/ros2_ws/src
cd ~/ros2_ws
colcon build
source install/setup.bash
