安装Go

官方安装指南

Windows

  1. 运行下载的 MSI 文件，按提示完成安装。 （安装后请关闭并重新打开命令提示符，以确保环境变量生效。）

  2. 验证安装：

    1. 打开“开始”菜单，输入 cmd 并回车。
    2. 在命令提示符中输入：
      go version
    3. 确认显示已安装的 Go 版本。
macOS

  1. 打开下载的包文件，按提示完成安装。 （安装后建议重启终端会话，以确保环境变量生效。）

  2. 验证安装：

    1. 打开终端，输入：
      go version
    2. 确认显示已安装的 Go 版本。
Linux

  1. 清理旧版本并解压新版本（请酌情使用 sudo）：

    rm -rf /usr/local/go && tar -C /usr/local -xzf go1.25.2.linux-amd64.tar.gz

  2. 配置环境变量： 将以下内容添加到 $HOME/.profile 或 /etc/profile 中：

    export PATH=$PATH:/usr/local/go/bin

    执行 source $HOME/.profile 使配置立即生效。

  3. 验证安装：

    go version

    确认显示已安装的 Go 版本。
